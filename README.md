## Project Setup

-   Create Virtual Environment

    -   On MacOS and Linux
        ```bash
        python3 -m venv .venv
        ```
    -   On Windows
        ```bash
        python -m venv .venv
        ```

-   Activate Virtual Environment

    -   On MacOS and Linux
        ```bash
        source .venv/bin/activate
        ```
    -   On Windows
        ```bash
        .venv\Scripts\activate.bat
        ```

-   Install Dependencies
    ```bash
    pip install -r requirements.txt
    ```
